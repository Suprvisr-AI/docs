# Changelog

A running log of new features, improvements, and fixes in Suprvisr AI.

---

## Upcoming Features

### Workspaces
Build project folders with shared AI context. Assign chats and documents, invite teammates to collaborate, and chat with the AI in a threaded shared environment. → [Learn more](/features/workspaces)

### Citations
Every response that draws from your internal data will surface Citation Cards — showing exactly which document and section was used. Explore sources without leaving the chat. → [Learn more](/features/citations)

### Dashboards
User and Admin dashboards showing usage frequency, estimated time saved, and common query topics across your team. → [Learn more](/features/dashboards)

### Self-Destructing Chats
Admin-controlled private chat mode — conversations not saved to history unless the user saves them. Timer icon beside New Chat. Off by default, enabled in Org Settings. → [Learn more](/features/self-destructing-chats)

### Branding & Prompts
Custom prompts for users, org-wide tone and brand settings for admins, plus an Assistant creator to build and share custom AI personas. → [Learn more](/features/branding-prompts)

### Reasoning & Verbosity
Multi-level reasoning for deeper, more confident answers. Verbosity modes (Quick / Balanced / Verbose) to control response depth. → [Learn more](/features/reasoning-verbosity)

### Dictate
Speak your messages using the mic icon. Voice-to-text input for hands-free prompting. (Note: text-to-speech AI voice dialogue coming in a future update.) → [Learn more](/features/dictate)

## March 2026

### Super Admin Dashboard

- Added `/super-admin` route gated to `@suprvisr.ai` email addresses
- **Overview**: 9 KPI cards — total tenants, users, DAU, messages today, active subscriptions with MRR, SharePoint files, uploads, conversations, and total storage
- **Trends**: 4 time-series charts (DAU, messages/day, new tenants/week, SharePoint files synced) with 7/30/60/90-day time range selector
- **Tenants table**: Sortable view of all orgs with per-tenant 30-day stats, subscription status, and MRR
- **Re-engagement panel**: Lists dormant tenants and users (inactive 7+ days or never used) with one-click "Copy email" for outreach
- **Mailing panel**: Listmonk subscriber count, list breakdown, and recent campaign stats (configure via env vars)

## February 2026

### Dice Prompt Feature

- Added a **dice icon** to the empty chat state
- Clicking the dice inserts a role-specific example prompt into the input field
- Prompts curated per role: Developer, Designer/Marketer, PM, Sales, Admin, and Other
- Prompts stored in the database — updated without code deploys
- Greeting in empty state now uses your preferred name from onboarding

---

### User Onboarding Flow

- Replaced modal-based onboarding with a **full-page onboarding experience**
- Updated welcome message to "Hey there, welcome to Suprvisr AI"
- **Step 1 — Welcome**: Animated greeting + preferred name input
- **Step 2 — Role Selection**: Developer, Product Manager, Designer, or Other
- **Step 3 — Features Showcase**: Tour of platform highlights
- Name and role saved to membership profile
- "Redo Onboarding" button added to Settings → Personalization
